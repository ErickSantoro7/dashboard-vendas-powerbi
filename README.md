# 📊 Dashboard de Vendas 2024 - Power BI

## 📌 Objetivo do Projeto
Este projeto tem como objetivo analisar o desempenho de vendas ao longo de 2024, identificando evolução mensal, produto líder e ranking de receita por produto.

---

## 🛠 Ferramentas Utilizadas
- SQL Server (tratamento e modelagem de dados)
- Power BI (visualização e construção do dashboard)
- DAX para cálculo de indicadores

---

## 📊 Indicadores Criados
- Total de Vendas
- Taxa de Cancelamento
- Produto Líder
- Receita do Produto Líder
- Produto Menos Vendido

---

## 📈 Análises Desenvolvidas
- Evolução mensal de receita
- Ranking de produtos por receita
- Identificação de sazonalidade

## 🎯 Principais Insights
- O mês de janeiro apresentou o maior volume de vendas.
- O Notebook Dell foi o produto líder em receita.
- Houve variação significativa entre fevereiro e março, indicando possível sazonalidade.
